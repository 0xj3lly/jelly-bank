# DSJAS default theme - *the new one*

This theme is a port of the Felicity Bank tool by 0xB9, located at <https://github.com/0xB9/Felicity-Bank-Inc>. It is a remake of a fake bank used in KitBoga live streams, called LR Jenkins Financial.

## Defining features

Clean and professional look

## Use case

* For those who have used this bank in the past and liked it
* For those who like KitBoga live streams and want to emulate his techniques
* Pretending to be an online only bank

## Possible giveaways

This theme's original source will be ranked high in search results if the name "Felicity Bank" is used. Therefore, we recommend **not using that name**. In addition, some parts of the site are remanent from KitBoga's jokes in his site. These could be giveaways.

Adding on to this, the fact that it is a remake of KitBoga's site may be a giveaway, as many scammers actually know about his site by now. We recommend that this theme is not used on suspicious scammers.

Finally, the bank contains the same level of fleshing out as the minimal default theme, with just a login and dashboard.
